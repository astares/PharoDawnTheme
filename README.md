### Install
```smalltalk
Metacello new 
	baseline: 'PharoDawnTheme';
	repository: 'github://sebastianconcept/PharoDawnTheme';
	load.
```

### For Pharo 7.0
```smalltalk
Metacello new 
	baseline: 'PharoDawnTheme';
	repository: 'github://sebastianconcept/PharoDawnTheme:pharo7';
	load.
```
